# 百度 OCR 快速开始

## 🚀 2 步开始使用

### 第一步：配置 Access Token

在项目根目录创建 `.env.local` 文件：

```env
NEXT_PUBLIC_BAIDU_OCR_ACCESS_TOKEN=483605608bc2d69ed9979463871dd4bc6095285a
```

### 第二步：重启并使用

```bash
# 重启开发服务器
pnpm dev
```

然后：
1. 访问 http://localhost:5000/admin/smart-import
2. 开启「OCR 识别」
3. 选择「百度 OCR API」
4. 上传图片
